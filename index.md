Den här sidan riktar sig till dig som drömmer om att bygga eller renovera ett hus eller fritidshus, och vill göra det hållbart. Här samlas perspektiv, goda exempel och regelmässiga möjligheter för ett sunt byggande. Målet är att göra genomtänkt byggande mer tillgängligt, för den som vill bygga bättre för både människa och miljö.

Här fylls det kontinuerligt på med information kring hur ett hållbart byggande kan se ut, inspiration från andra projekt och framförallt lösningar på byggtekniska utmaningar. Målet är att ge vägledning om hur bostäder kan utformas för att vara miljömässsigt, samhällsmässigt och ekonomisktt mer hållbara än det konventionella alternativet – oavsett om det handlar om att ta hand om det befintliga hus, bygga nytt eller bygga annorlunda.
