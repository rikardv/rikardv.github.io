Den här sidan riktar sig till dig som drömmer om att bygga eller renovera ett hus eller fritidshus, och vill göra det hållbart. Här samlas perspektiv och goda exempel på sunt byggande. Målet är att göra det enklare att bygga och renovera på ett genomtänkt sätt, på så vis sänka trösklarna för ett mer hållbart byggande av småhus.

Här fylls det kontinuerligt på med information kring hur ett hållbart byggande kan se, inspiration från andra projekt och framförallt lösningar på byggtekniska utmaningar. Målet är att ge vägledning om hur bostäder kan utformas för att vara miljömässsigt, samhällsmässigt och ekonomisktt mer hållbara än det konventionella alternativet – oavsett om det handlar om att ta hand om det befintliga hus, bygga nytt eller bygga annorlunda.
