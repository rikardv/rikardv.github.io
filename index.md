Den här sidan riktar sig till dig som drömmer om att bygga eller renovera ett hus eller fritidshus, och vill göra det hållbart. Här samlar jag perspektiv, goda exempel och regelmässiga möjligheter för ett sunt byggande. Målet är att göra genomtänkt byggande mer tillgängligt, för den som vill bygga bättre för både människa och miljö.

Här fylls det kontinuerligt på med information kring hur ett hållbart byggande kan se ut med inspiration från andra projekt och hur typlösningar på hållbarat byggande, något som saknas i branchen idag. Målet är att ge vägledning om hur bostäder kan utformas för att vara miljömässsigt, samhällsmässigt och ekonomisktt mer hållbara än det konventionella alternativet – oavsett om det handlar om ett befintligt hus, att bygga nytt eller att bygga annorlunda.
